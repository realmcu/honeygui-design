# ChatBotTemplate

Created from template: **ChatBot**

AI chat interface with message bubbles and input area.

## Project Info
- APP ID: com.honeygui.chatbot
- Resolution: 480X320
- Target Engine: HoneyGUI
- Min SDK: API 2: HoneyGUI V1.1.0
- Pixel Mode: ARGB8888

## Features
- Header with title
- Chat message area (bot and user messages)
- Input field with send button
- Clean, modern UI design

## Getting Started

1. Open the HML file in `ui/main/ChatBotMain.hml`
2. Design your UI in the visual designer
3. Generate code and compile
