# 图片资源转换方案

## 背景

HoneyGUI 引擎无法直接显示 PNG/JPG 图片，需要转换为特定的 bin 格式。当前 SDK 仅提供 Windows exe 转换工具，不兼容 Linux/macOS，影响 VSCode 插件的跨平台使用。

## 问题分析

### 当前流程（存在问题）

```
设计器拖拽图片 (PNG/JPG)
        ↓
组件引用原始图片路径
        ↓
生成 C 代码
        ↓
编译仿真
        ↓
❌ HoneyGUI 无法识别 PNG/JPG
```

### 目标流程

```
设计器拖拽图片 (PNG/JPG)
        ↓
【新增】图片转换 (PNG/JPG → bin)
        ↓
组件引用 bin 文件
        ↓
生成 C 代码
        ↓
编译仿真
        ↓
✅ 正常显示
```

## 方案选型

### 实现语言对比

| 方案 | 跨平台 | 实现难度 | 依赖 | 备注 |
|------|--------|----------|------|------|
| Windows exe（当前） | ❌ 仅 Windows | - | 外部工具 | 不可接受 |
| TypeScript + sharp | ✅ | 中 | npm 原生模块 | sharp 有平台兼容问题 |
| **Python + Pillow** | ✅ | 低 | Python 环境 | **推荐** |

### 选择 Python 的理由

1. **用户已有 Python 环境** - 编译仿真依赖 SCons，SCons 需要 Python
2. **Pillow 生态成熟** - 图像处理功能强大，跨平台稳定
3. **开发效率高** - 图像格式转换逻辑用 Python 更直观
4. **可独立使用** - 也可作为命令行工具单独调用

### 代码归属对比

| 方面 | 放在 SDK | 放在插件 |
|------|----------|----------|
| 版本同步 | ✅ 格式变化时自动同步 | ❌ 需手动跟进 |
| 复用性 | ✅ 命令行/其他工具可用 | ❌ 仅插件可用 |
| 维护责任 | SDK 团队 | 插件团队 |
| 格式兼容 | ✅ 保证与引擎一致 | ⚠️ 可能滞后 |

### 决策：放在 HoneyGUI SDK

**理由**：
- 图片 bin 格式由 SDK 定义，转换工具应与格式定义在一起
- 格式升级时，转换工具和引擎同步更新
- 不使用插件的开发者也能使用

### 插件调用示例

```typescript
// src/assets/ImageConverterService.ts
import { spawn } from 'child_process';
import * as path from 'path';

export class ImageConverterService {
    constructor(private sdkPath: string) {}

    async convert(inputPath: string, outputPath: string, options?: ConvertOptions): Promise<void> {
        const script = path.join(this.sdkPath, 'tools', 'image_converter.py');
        
        return new Promise((resolve, reject) => {
            const args = [script, '-i', inputPath, '-o', outputPath];
            if (options?.format) args.push('-f', options.format);
            
            const proc = spawn('python', args);
            proc.on('exit', code => code === 0 ? resolve() : reject(new Error(`转换失败: ${code}`)));
        });
    }

    async convertProjectAssets(projectRoot: string): Promise<string[]> {
        // 扫描 assets 目录，批量转换
    }
}
```

### 修改 SimulationRunner

```typescript
// src/simulation/SimulationRunner.ts
async run(): Promise<void> {
    // 1. 环境检查
    await this.checkEnvironment();
    
    // 2. 【新增】图片资源转换
    await this.convertAssets();
    
    // 3. 代码生成
    await this.generateCode();
    
    // 4. 编译
    await this.compile();
    
    // 5. 运行仿真
    await this.runSimulator();
}

private async convertAssets(): Promise<void> {
    const converter = new ImageConverterService(this.sdkPath);
    const convertedFiles = await converter.convertProjectAssets(this.projectRoot);
    this.logger.log(`转换了 ${convertedFiles.length} 个图片资源`);
}
```
