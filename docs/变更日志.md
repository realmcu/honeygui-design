# 变更日志

## [2025-11-25] v1.1.9

### 新增
- 支持从资源预览面板拖拽图片到画布创建组件

### 改进
- 统一图片组件创建逻辑，提取 `createImageComponentAtPosition` 函数
- 添加详细的拖拽和删除调试日志

### 修复

- 修复 webview 中 confirm 对话框不可用的问题
- 修复删除资源时路径解析错误（webview URI vs 文件系统路径）

### 改进
- 创建 findDropTarget 辅助函数统一查找目标容器
- 多图片创建组件时自动偏移位置（每个偏移 20px）
