# 变更日志

## [Unreleased]

### 新增功能

- **组件树拖拽**
  - 支持在组件树中拖拽组件改变父子关系
  - 支持在兄弟组件之间插入调整顺序
  - 三种放置方式：插入前、插入后、作为子组件
  - 智能识别鼠标位置显示不同的放置指示器
  - hg_view 不能拖拽（只能作为根组件）
  - hg_list_item 不能拖拽（由 list 控件自动管理）
  - 只有容器控件（hg_view, hg_window, hg_list_item）可以接收子组件
  - 自动验证拖拽规则，防止循环引用
  - 修复：拖动后立即刷新组件树显示
