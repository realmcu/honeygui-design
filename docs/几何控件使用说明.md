# 几何控件使用说明

本文档介绍 HoneyGUI Design 中的几何控件（圆弧和圆角矩形）的使用方法。

## 概述

几何控件位于组件库的"图形"分类中，包括：
- **hg_arc**: 圆弧控件
- **hg_rect**: 圆角矩形控件

## 圆弧控件 (hg_arc)

### 基本信息
- **图标**: 🌙
- **默认尺寸**: 100 x 100
- **底层类型**: `gui_arc_t`
- **创建函数**: `gui_arc_create`
- **头文件**: `gui_arc.h`

### 属性说明

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| radius | number | 40 | 圆弧半径（像素） |
| startAngle | number | 0 | 起始角度（度数，0° 为右侧） |
| endAngle | number | 270 | 结束角度（度数） |
| strokeWidth | number | 8 | 线条宽度（像素） |
| color | color | #007acc | 圆弧颜色 |
| opacity | number | 255 | 透明度（0-255，0 为完全透明，255 为完全不透明） |

### 使用示例

#### 1. 进度环
```xml
<hg_arc id="progress_ring" 
        x="50" y="50" 
        radius="40" 
        startAngle="0" 
        endAngle="270" 
        strokeWidth="8" 
        color="#00ff00"
        opacity="255" />
```

#### 2. 仪表盘指示器
```xml
<hg_arc id="gauge_indicator" 
        x="100" y="100" 
        radius="60" 
        startAngle="135" 
        endAngle="225" 
        strokeWidth="12" 
        color="#ff6600"
        opacity="200" />
```

#### 3. 半透明圆弧
```xml
<hg_arc id="transparent_arc" 
        x="150" y="150" 
        radius="50" 
        startAngle="0" 
        endAngle="180" 
        strokeWidth="10" 
        color="#007acc"
        opacity="128" />
```

#### 4. 完整圆环（360度）
```xml
<hg_arc id="full_ring" 
        x="200" y="200" 
        radius="70" 
        startAngle="0" 
        endAngle="360" 
        strokeWidth="15" 
        color="#ff0000"
        opacity="255" />
```

**注意**：
- 完整圆环（startAngle=0, endAngle=360）使用优化的绘制算法
- 如果完整圆环不显示，请检查：
  1. 圆心坐标是否在父容器范围内
  2. 半径 + 线宽是否超出父容器边界
  3. 透明度设置是否正确

### 生成的 C 代码

```c
// 创建圆弧（颜色使用完全不透明，透明度通过 opacity_value 控制）
progress_ring = (gui_obj_t *)gui_arc_create(parent, "progress_ring", 
                                            50, 50,           // 中心坐标
                                            40,               // 半径
                                            0, 270,           // 起始和结束角度
                                            8,                // 线宽
                                            (gui_color_t){.color.argb_full = 0xFF00FF00});  // 颜色（alpha=0xFF，完全不透明）

// 半透明圆弧示例（通过 opacity_value 控制透明度）
transparent_arc = (gui_obj_t *)gui_arc_create(parent, "transparent_arc", 
                                              150, 150, 
                                              50, 
                                              0, 180, 
                                              10, 
                                              (gui_color_t){.color.argb_full = 0xFF007ACC});  // 颜色（alpha=0xFF）
((gui_arc_t *)transparent_arc)->opacity_value = 128;  // 设置透明度为 128
```

**重要说明**：
- 圆弧控件的透明度通过 `opacity_value` 字段控制，而不是颜色的 alpha 通道
- 颜色的 alpha 应该始终设置为 0xFF（255，完全不透明）
- 这样可以避免双重透明度应用，确保透明度效果正确

**注意**：圆弧的 x, y 坐标是圆心位置，SDK 会自动计算边界框。

## 圆角矩形控件 (hg_rect)

### 基本信息
- **图标**: ▭
- **默认尺寸**: 120 x 80
- **底层类型**: `gui_rounded_rect_t`
- **创建函数**: `gui_rect_create`
- **头文件**: `gui_rect.h`

### 属性说明

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| borderRadius | number | 0 | 圆角半径（像素，0 表示直角矩形） |
| fillColor | color | #007acc | 填充颜色 |
| opacity | number | 255 | 透明度（0-255，0 为完全透明，255 为完全不透明） |

### 使用示例

#### 1. 普通矩形
```xml
<hg_rect id="rect1" 
         x="10" y="10" 
         w="120" h="80" 
         fillColor="#007acc" 
         opacity="255" />
```

#### 2. 圆角矩形
```xml
<hg_rect id="rounded_rect" 
         x="150" y="10" 
         w="120" h="80" 
         borderRadius="10" 
         fillColor="#ff6600" 
         opacity="255" />
```

#### 3. 半透明背景
```xml
<hg_rect id="overlay" 
         x="0" y="0" 
         w="480" h="272" 
         fillColor="#000000" 
         opacity="128" />
```

### 生成的 C 代码

```c
// 创建圆角矩形（所有参数在创建时传入，包括透明度）
rounded_rect = (gui_obj_t *)gui_rect_create(parent, "rounded_rect", 
                                            150, 10,          // 位置
                                            120, 80,          // 尺寸
                                            10,               // 圆角半径
                                            (gui_color_t){.color.argb_full = 0xFFFF6600});  // 颜色（ARGB 格式）

// 半透明遮罩层示例（alpha = 128）
overlay = (gui_obj_t *)gui_rect_create(parent, "overlay", 
                                       0, 0, 
                                       480, 272, 
                                       0, 
                                       (gui_color_t){.color.argb_full = 0x80000000});  // alpha = 0x80 (128)

// 如果需要后续修改属性，可以使用：
// gui_rect_set_radius(rounded_rect, 15);
// gui_rect_set_color(rounded_rect, new_color);
```

**颜色格式说明**：
- ARGB 格式：`0xAARRGGBB`
- AA = Alpha（透明度，00=完全透明，FF=完全不透明）
- RR = Red（红色）
- GG = Green（绿色）
- BB = Blue（蓝色）

## 使用场景

### 圆弧控件适用场景
- 进度指示器（环形进度条）
- 仪表盘刻度
- 圆形菜单
- 装饰性图形元素
- 加载动画

### 圆角矩形控件适用场景
- 背景色块
- 卡片背景
- 按钮背景
- 遮罩层
- 分隔线
- 对话框背景

## 注意事项

1. **父容器要求**: 几何控件必须放置在容器控件（`hg_view` 或 `hg_window`）内部

2. **坐标系统**: 
   - **圆弧**: x, y 是圆心坐标，SDK 会自动计算边界框
   - **矩形**: x, y 是左上角坐标
   - **重要**：圆弧的边界框 = 圆心 ± (半径 + 线宽/2 + 2)，确保不超出父容器

3. **角度系统**: 
   - 圆弧使用度数（0-360），0° 为右侧，顺时针方向
   - 完整圆环：startAngle=0, endAngle=360
   - 跨越 0° 的圆弧：例如 startAngle=350, endAngle=10

4. **颜色格式**: 
   - 支持十六进制颜色：`#RRGGBB` 或 `#RRGGBBAA`（带透明度）
   - 会自动转换为 `gui_color_t` 结构体
   - 透明度可以通过颜色的 alpha 通道设置，也可以通过独立的 `opacity` 属性设置
   - `opacity` 属性会覆盖颜色中的 alpha 值（范围 0-255）

5. **性能优化**:
   - 圆弧使用缓存机制，只在参数变化时重新渲染
   - 矩形对大尺寸和透明度有不同的渲染策略
   - 支持旋转、缩放、平移等变换

6. **事件支持**: 两个控件都支持点击事件（`onClick`）

## 底层 SDK 依赖

确保你的 HoneyGUI SDK 包含以下文件：
- `gui_arc.h` / `gui_arc.c`
- `gui_rect.h` / `gui_rect.c`

这些文件位于 SDK 的 `realgui/widget/gui_geometry` 目录下。

## 高级功能

### 变换支持

两个控件都支持以下变换操作：

```c
// 旋转
gui_arc_rotate(arc, 45.0f);        // 旋转 45 度
gui_rect_rotate(rect, 30.0f);      // 旋转 30 度

// 缩放
gui_arc_scale(arc, 1.5f, 1.5f);    // 放大 1.5 倍
gui_rect_scale(rect, 0.8f, 1.2f);  // X 缩小，Y 放大

// 平移
gui_arc_translate(arc, 10, 20);    // 向右 10，向下 20
gui_rect_translate(rect, -5, 15);  // 向左 5，向下 15
```

这些变换可以组合使用，实现复杂的视觉效果。
