# 项目模板功能

## 概述

从 v1.3.2 开始，HoneyGUI Design 支持从预定义模板创建项目，帮助用户快速启动常见的 GUI 应用场景。

## 模板系统架构

### 目录结构
```
src/template/templates/
├── ITemplate.ts              # 模板接口
├── BaseTemplate.ts           # 模板基类
├── index.ts                  # 注册中心
├── wearable/                 # 可穿戴设备
│   └── smartwatch/
│       ├── SmartWatchTemplate.ts
│       └── assets/           # 模板资源
├── utility/                  # 工具类
│   ├── settings/
│   └── dashboard/
└── README.md                 # 开发指南
```

### 设计特点
- **模块化**：每个模板独立管理
- **资源隔离**：模板资源与代码放在一起
- **易扩展**：添加新模板只需创建新目录
- **类型安全**：完整的 TypeScript 类型定义

## 可用模板

### 1. Smart Watch UI (智能手表界面)
- **适用场景**: 可穿戴设备、智能手表
- **推荐分辨率**: 480X272
- **包含组件**:
  - 时间显示 (大字号文本)
  - 日期显示 (小字号文本)
  - 电池电量进度条

### 2. Settings Menu (设置菜单)
- **适用场景**: 系统设置、配置界面
- **推荐分辨率**: 480X272
- **包含组件**:
  - 标题文本
  - WiFi 开关选项
  - 蓝牙开关选项
  - 亮度滑块

### 3. Dashboard (仪表盘)
- **适用场景**: 数据监控、状态展示
- **推荐分辨率**: 800X480
- **包含组件**:
  - 标题文本
  - CPU 使用率卡片 (文本 + 进度条)
  - 内存使用卡片 (文本 + 进度条)
  - 状态文本

## 使用方法

### 1. 打开创建项目面板
```
Ctrl+Shift+P → HoneyGUI: New Project
```

### 2. 切换到模板 Tab
点击 "Create template project" 标签页

### 3. 选择模板
- 浏览可用模板卡片
- 点击选择你需要的模板
- 模板会自动设置推荐分辨率

### 4. 填写项目信息
- **Project name**: 项目名称
- **Save location**: 保存位置
- **APP ID**: 应用标识符
- **Resolution**: 分辨率（已自动设置为推荐值）
- **Minimum SDK**: 最低 SDK 版本
- **Pixel Mode**: 像素模式
- **HoneyGUI SDK Path**: SDK 路径（可选）

### 5. 创建项目
点击 "Create from Template" 按钮

## 项目结构

模板项目的结构与空项目相同：

```
my-project/
├── ui/main/
│   └── MyProjectMain.hml    # 包含模板预设的组件
├── src/                      # 生成的代码目录
├── assets/                   # 资源文件
├── project.json              # 项目配置（包含 template 字段）
└── README.md                 # 项目说明
```

## 自定义模板

模板创建后，你可以：
- 在设计器中修改组件属性
- 添加或删除组件
- 调整布局
- 生成代码并编译

模板只是提供初始结构，后续完全可以自由修改。

## 技术实现

### 模板接口
```typescript
export interface ITemplate {
    id: string;
    name: string;
    description: string;
    category: string;
    recommendedResolution: string;
    
    generateHml(config: TemplateConfig): string;
    getAssets(): TemplateAsset[];
    copyAssets(projectPath: string): Promise<void>;
    getPreviewImage(): string | null;
}
```

### 添加新模板

详见 [模板开发指南](../src/template/templates/README.md)

简要步骤：
1. 在合适的分类目录创建模板类
2. 继承 `BaseTemplate` 并实现 `generateHml()`
3. 在 `index.ts` 注册模板
4. 编译：`npm run compile`
