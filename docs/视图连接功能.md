# 视图连接功能

## 功能概述

在设计器画布上可视化显示 `hg_view` 组件之间的导航关系，通过箭头连接表示 `view_switch` 配置。

## 需求说明

### 核心需求

1. **可视化连接**
   - 在画布上绘制箭头，连接有 `view_switch` 关系的 hg_view
   - 箭头从源 view 边缘指向目标 view 边缘
   - 箭头上显示事件类型（LEFT/RIGHT/UP/DOWN）
   - 支持显示/隐藏切换

2. **数据关系**
   - `view_switch` 配置在 hg_view 组件中
   - `target` 字段指向目标 view 的 **name**（不是 id）
   - 支持一个 view 有多个 view_switch（多个导航方向）

3. **交互需求**
   - 工具栏按钮切换显示/隐藏
   - 箭头随画布缩放和平移
   - 拖拽创建新连接（TODO）

### 边缘点计算

```typescript
// 1. 计算中心点
const fromCenter = {
  x: from.position.x + from.position.width / 2,
  y: from.position.y + from.position.height / 2
};

// 2. 计算角度
const angle = Math.atan2(dy, dx);

// 3. 计算边缘点
const fromEdge = {
  x: fromCenter.x + Math.cos(angle) * (from.position.width / 2),
  y: fromCenter.y + Math.sin(angle) * (from.position.height / 2)
};

// 4. 边界修正
fromEdge.x = Math.max(from.position.x, 
  Math.min(from.position.x + from.position.width, fromEdge.x));
```
