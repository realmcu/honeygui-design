# 时间接口使用说明

## 功能概述

为 `hg_label` 组件添加了自动更新时间的功能，可以在设计器中配置时间格式，生成的代码会自动创建时间更新回调函数。

## 使用方法

### 1. 在设计器中配置

1. 在画布中添加一个 `hg_label` 组件
2. 在右侧属性面板的"数据"分组中，找到"时间格式"选项
3. 从下拉列表中选择需要的时间格式：
   - `HH:mm:ss` - 时分秒（例如：14:30:25）
   - `HH:mm` - 时分（例如：14:30）
   - `YYYY-MM-DD` - 年月日（例如：2024-12-24）
   - `YYYY-MM-DD HH:mm:ss` - 完整日期时间（例如：2024-12-24 14:30:25）
   - `MM-DD HH:mm` - 月日时分（例如：12-24 14:30）
   - 留空 - 不启用时间更新功能

### 2. 生成代码

选择"生成代码"后，系统会自动生成：

#### UI 代码（`*_ui.c`）
```c
// 在 view 的 switch_in 回调中创建定时器
static void mainView_switch_in(gui_view_t *view)
{
    GUI_UNUSED(view);
    
    // 创建组件...
    hg_label_xxx = gui_text_create((gui_obj_t *)view, "time_label", 10, 10, 100, 24);
    gui_text_content_set((gui_text_t *)hg_label_xxx, (void *)"00:00:00", 8);
    
    // 创建时间更新定时器（绑定到 label 对象）
    gui_obj_create_timer(GUI_BASE(hg_label_xxx), 1000, true, hg_label_xxx_time_update_cb);
}
```

#### 回调代码（`*_callbacks.c`）
```c
void hg_label_xxx_time_update_cb(void *p)
{
    GUI_UNUSED(p);
    
    time_t now = time(NULL);
    struct tm *t = localtime(&now);
    static char time_str[9];
    
    sprintf(time_str, "%02d:%02d:%02d", t->tm_hour, t->tm_min, t->tm_sec);
    
    gui_text_content_set((gui_text_t *)hg_label_xxx, (void *)time_str, strlen(time_str));
}
```

### 3. 定时器机制

生成的代码使用 SDK 的 `gui_obj_create_timer()` 函数创建定时器：

```c
gui_obj_create_timer(GUI_BASE(label_obj), interval, reload, callback);
```

参数说明：
- `obj`：GUI 对象指针（绑定到 label 对象）
- `interval`：定时器间隔（毫秒）
- `reload`：是否自动重载（true=循环，false=单次）
- `callback`：回调函数指针 `void (*callback)(void *)`

**优点**：
- 自动创建定时器，无需手动配置
- 定时器在 view 切入时自动创建
- 每个时间 label 有独立的定时器，互不干扰
- 定时器绑定到 label 对象，生命周期由 label 管理
- 简单可靠，基于对象的定时器管理

**依赖的 SDK API**：
- `gui_obj_create_timer()`：创建定时器
- `gui_obj_timer_handler()`：SDK 内部自动调用，处理定时器触发

## 技术实现

### 数据模型
在 `ComponentData` 接口中添加了 `timeFormat` 字段：
```typescript
export interface ComponentData {
  timeFormat?: string;  // 时间格式（用于 label 组件自动更新时间）
  // ...
}
```

### 代码生成
- `LabelGenerator.ts`：在属性设置时检查 `timeFormat`，如果存在则使用时间格式占位符
- `ViewGenerator.ts`：在 view 的 `switch_in` 回调中，为所有带时间格式的 label 创建定时器
- `CallbackFileGenerator.ts`：收集所有带时间格式的 label，生成对应的时间更新回调函数

### 回调函数特点
- 使用标准 C 库的 `time()` 和 `localtime()` 函数获取系统时间
- 使用 `static` 变量存储时间字符串，避免频繁分配内存
- 根据不同的时间格式生成不同的 `sprintf` 格式化代码
- 自动计算所需的缓冲区大小

### 定时器创建
在 view 的 `switch_in` 回调中自动创建定时器：
```c
gui_obj_create_timer(GUI_BASE(hg_label_xxx), 1000, true, hg_label_xxx_time_update_cb);
```

定时器绑定到 label 对象，当 view 切入时自动创建。每个时间 label 有独立的定时器，避免冲突。SDK 会自动调用 `gui_obj_timer_handler()` 处理定时器触发，每秒调用一次回调函数更新时间。

## 注意事项

1. **系统时间依赖**：此功能依赖于系统的 `time()` 函数，确保目标平台支持标准 C 时间库
2. **SDK API 依赖**：需要 SDK 提供 `gui_obj_create_timer()` 函数用于创建定时器
3. **自动触发**：定时器在 view 切入时自动创建并启动，无需手动配置
4. **性能优化**：每个时间 label 使用独立定时器，按需更新
5. **保护区**：生成的回调代码在 `*_callbacks.c` 中，使用保护区机制，重新生成时会保留用户修改

## 示例

### 创建一个数字时钟
1. 添加一个 `hg_label` 组件
2. 设置时间格式为 `HH:mm:ss`
3. 设置字体大小为 32
4. 设置对齐方式为 `CENTER`
5. 生成代码并编译
6. 定时器会自动创建，每秒更新一次时间

### 创建日期显示
1. 添加一个 `hg_label` 组件
2. 设置时间格式为 `YYYY-MM-DD`
3. 生成代码并编译
4. 定时器会自动创建，每秒更新一次日期

### 多个时间控件
如果界面中有多个时间 label，每个都会自动创建独立的定时器，互不干扰。
