# 已知问题

## ✅ 已解决的问题

### 左侧面板折叠功能异常（已解决）

**状态**: ✅ 已通过 Tab 切换功能解决

**原问题描述**:
1. 组件库和控件树折叠时，不能正确释放空间，框反而变大
2. 资源预览展开时，只能看到顶部的红色边框，内容被压缩到看不见
3. flex布局计算异常

**解决方案**: 
改用 Tab 切换模式，不再使用折叠功能。详见 [Tab切换功能.md](./Tab切换功能.md)

## 🔄 待解决的问题

### 协同开发断线重连

**问题描述**: 断线后需手动重新加入会话，没有自动重连机制。

**影响**: 网络不稳定时需要频繁手动重连。

**计划**: 后续版本实现自动重连。

---

### 大型组件树性能

**问题描述**: 100+ 组件时，画布渲染和操作可能变慢。

**影响**: 复杂界面设计时体验下降。

**计划**: 使用虚拟化渲染优化。
