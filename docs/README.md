- ✅ 仿真器改用 Terminal 运行，支持 letter shell 交互输入

- 📖 详见：[预览和编译架构.md](./预览和编译架构.md)

- 📖 详见：[项目模板功能.md](./项目模板功能.md)

- ✅ 组件库、资源、组件树三个 Tab
- 📖 详见：[Tab切换功能.md](./Tab切换功能.md)

### 2025-12-02 - 视图连接功能
- ✅ 实现 hg_view 导航关系可视化
