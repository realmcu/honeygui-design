1. **项目介绍** - 从根目录的 [README.md](../README.md) 开始
2. **开发指南** - 阅读 [DEVELOPMENT.md](../DEVELOPMENT.md) 了解项目架构
3. **调试速查表** - 收藏 [调试速查表](./debugging/DEBUG_QUICK_REF.md) 以备不时之需
