## 🚀 快速开始

### 新用户推荐阅读顺序

1. **安装后第一步**：浏览 [功能使用手册](./功能使用手册.md) 了解基本操作
2. **转换资源时**：参考 [资源转换指南](./资源转换指南.md)
3. **部署到设备**：查看 [嵌入式部署](./嵌入式部署方案.md)
4. **参与开发**：阅读 [开发指南](./开发指南.md)

### 开发者推荐阅读顺序

1. **开始开发**：[开发指南](./开发指南.md) → 架构设计 → 开发命令
2. **添加功能**：[开发指南](./开发指南.md) → 开发流程 → 添加新组件/模板
3. **运行测试**：[开发指南](./开发指南.md) → 测试指南
4. **调试问题**：[开发指南](./开发指南.md) → 调试技巧

## 📖 文档说明

### 文档组织原则

本文档系统遵循以下原则：
- **用户视角**：按功能和使用场景组织，而非按代码结构
- **减少碎片化**：相关主题合并到同一文档中
- **快速查找**：使用目录和清晰的标题层级
- **保持更新**：文档与代码同步更新
